Why the hosted apps instead of my single-file OpenCascade build?
A true B-Rep kernel is an ~11 MB WebAssembly module. Loading it from a CDN inside a single local file is fragile — under file:// the browser often blocks or stalls the wasm fetch (that's the "keeps loading" you hit). Chili3d, Replicad and CascadeStudio are served over https with the wasm packaged correctly, so they load reliably. Same kernel, none of the loading headaches.

Tip: for the offline tools, if a link doesn't open, double-click the .html file in your folder instead.
